# Tools

## InstallerBuilder

Usage :

```bash
python ./tools/installer_builder.py
```

## Packer

**Usage** :

```bash
python ./tools/minifier.py
```

## Minify

**Requirement** : 

- [luamin](https://github.com/mathiasbynens/luamin) installed and available in your shell paths.

**Usage** :

```bash
luamin -f ccmaze.lua > ccmaze-min.lua
```

## Build

**Usage** :
